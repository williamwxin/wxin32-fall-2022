The primary packages to install are seaborn, sklearn, torch, and matplotlib

I used my GPU for the torch sections, but any mentions of 'cuda' or 'device' can be removed to have it work with CPU only.

The wine dataset can be found here:
https://www.kaggle.com/datasets/dell4010/wine-dataset
This is just a source that combined both red and white wines. Related info can be found here:
http://www3.dsi.uminho.pt/pcortez/wine/
https://archive.ics.uci.edu/ml/datasets/wine+quality

The article dataset can be found here:
https://archive.ics.uci.edu/ml/datasets/Online+News+Popularity

Running the python scripts will generate all necessary graphs
